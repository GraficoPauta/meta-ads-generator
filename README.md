# 🚀 Meta Ads Campaign Generator

Generador automático de campañas Meta Ads con IA.

## ✨ Features
- Generar 4 copies automáticos desde landing
- Crear campañas en Meta automáticamente (BORRADOR)
- 4 ads listos para cargar creatives
- Sin instalación para usuarios
